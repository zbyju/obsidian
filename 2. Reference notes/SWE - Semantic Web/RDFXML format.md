RDF/XML is a format that utilizes XML for representing RDF graphs.

```xml
<rdf:RDF xmlns:rdf="http://www.w3.org/1999/02/22-rdf-syntax-ns#">
	<rdf:Description rdf:about="http://example.com/item">
		<title>Example Item</title>
	</rdf:Description>
</rdf:RDF>
```
